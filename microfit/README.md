# MicroFit - Fitness Tracking System

MicroFit is a full-stack fitness management application that helps users monitor workouts, track weight progress, calculate BMI, and analyze fitness data.

## Features

* User Registration & Login
* JWT Authentication
* Workout Management
* Weight Tracking
* BMI Calculator
* Dashboard Analytics
* Fitness Progress Monitoring

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* JPA/Hibernate
* Maven
* REST APIs

### Database

* MySQL

### Frontend

* HTML
* CSS
* JavaScript
