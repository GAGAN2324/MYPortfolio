# Mutual Funds Performance Prediction

A full-stack web application for analyzing and predicting mutual fund performance using Machine Learning algorithms.

## Features

* Mutual Fund Performance Prediction
* SIP Calculator
* AMC Selection
* Fund Comparison
* Algorithm Comparison Dashboard
* Data Visualization using Charts
* Exploratory Data Analysis (EDA)
* AI Insights Dashboard

## Machine Learning Algorithms

* Linear Regression
* Random Forest
* ARIMA

## Tech Stack

### Backend

* Java
* Spring Boot
* Maven
* REST APIs

### Frontend

* React.js
* Vite
* JavaScript
* HTML5
