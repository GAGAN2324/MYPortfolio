# MYPortfolio

A collection of full-stack Java projects developed using Spring Boot, React.js, MySQL, Machine Learning, and REST APIs.

## Projects

### 1. Mutual Funds Performance Prediction

A full-stack web application that analyzes and predicts mutual fund performance using Machine Learning algorithms.

**Features**

* Mutual Fund Analysis
* Performance Prediction
* SIP Calculator
* Data Visualization
* Fund Comparison
* Algorithm Comparison Dashboard

**Tech Stack**

* Java
* Spring Boot
* React.js
* Vite
* Machine Learning
* Chart.js
* Axios
* REST APIs
* Maven
* Git & GitHub
